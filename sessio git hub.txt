Create a new project in intellij idea	1.1 Select JAVA project.	1.2 Select JAVA JDK 1.8 home, if you dont have one, intellij idea will do it for you by selecting specific version and from where to download it.	1.3 Provide the project name and finish it.2. Setting up repo	3.1 Setup your local repo by using git init. 	3.2 And create a remote repo on github. 	3.3 Add remote to your local repo using git remote add origin <URL> 	3.4 Commit your changes and push it on github.
